# Airline Status Calculator

A C++ application that determines airline loyalty status based on flight segments and miles traveled.

## Features

- Accepts user input for miles and flight segments
- Validates user input
- Determines membership tier automatically
- Supports multiple loyalty levels:
  - Platinum
  - Gold
  - Silver
  - Member

## Technologies

- C++
- Conditional Logic
- User Input Handling
- Data Validation
